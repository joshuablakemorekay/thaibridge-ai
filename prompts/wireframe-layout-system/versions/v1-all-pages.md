# v1 — "apply the wireframe to all pages?"

```
Should the wireframe layout be applied to all pages? If yes. Instruction: Add the wireframe layout to all pages.
```

**What this version got wrong:** it treated the wireframe as a single layout to stamp on every page. That would make simple info pages (About, Contact) and full-width tools (the AI chat, the dictionary) look wrong — the 3-column shape only fits reference/content pages. The fix was to *not* force one layout everywhere.
