3. **Install package:**
   ```bash
   pip install anthropic --break-system-packages
   ```

4. **Get API key:**
   - Visit: https://console.anthropic.com
   - Create key

5. **Set API key:**
   ```bash
   export ANTHROPIC_API_KEY='sk-ant-your-key'
   ```

6. **Run it:**
   ```bash
   python app.py
   ```
